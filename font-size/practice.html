フォントサイズの比較

親要素: フォントサイズ20pxは #parent { font-size: 20px; /* 親のフォントサイズを20pxに設定 */ }

子要素 (em): 親（20px）の1.5倍の30pxにしたい場合、font-size: 1.5em

子要素 (rem): ルート（16px）の1.5倍の24pxにしたい場合、font-size: 1.5rem

子要素 (%): 親（20px）の150%の30pxにしたい場合、font-size: 150%

子要素 (vw): ビューポート幅（1029px）の2% 20.58pxにしたい場合、font-size: 2vw

子要素 (vh): ビューポート高さ（678ppx）の2% 13.56pxにしたい場合、font-size: 2vh

body { font-size: 16px; /* 基準となるフォントサイズ */ }
#parent { font-size: 20px; /* 親のフォントサイズを20pxに設定 */ }
#child-em { font-size: 1.5em; /* 親のフォントサイズの1.5倍 */ }
#child-rem { font-size: 1.5rem; /* HTMLのルートフォントサイズの1.5倍 */ }
#child-percent { font-size: 150%; /* 親のフォントサイズの150% */ }
#child-vw { font-size: 2vw; /* ビューポート幅（1029px）の2％に基づいている */ }
#child-vh { font-size: 2vh; /* ビューポート高さの2％（678px）に基づいている */ }
