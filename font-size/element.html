要素サイズの比較

em: 幅10em, 高さ5em
rem: 幅10rem, 高さ5rem
％: 幅50％, 高さ50%
vw: 幅10vw, 高さ5vw
vh: 幅10vh, 高さ5vh

body { width: 100vw; /* ビューポート幅を基準に */ height: 100vh; /* ビューポート高さを基準に */そのため、ビューポートいっぱいになっています。 }
#container { width: 400px; /* 固定幅の親要素 */ height: 200px; /* 固定高さの親要素 */ background-color: lightgray; padding: 10px;これが親要素 }
.child-em { width: 10em; /* 親のフォントサイズに基づく */16pxの10倍、160px height: 5em;16pxの5倍、80px background-color: lightcoral; }
.child-rem { width: 10rem; /* ルートフォントサイズに基づく */ height: 5rem; background-color: lightblue; }
.child-percent { width: 50%; /* 親の幅の50% */ height: 50%; /* 親の高さの50% */ background-color: lightgreen; }
.child-vw { width: 10vw; /* ビューポート幅の10% */ height: 5vw; /* ビューポート幅の5% */ background-color: lightpink; }
.child-vh { width: 10vh; /* ビューポート高さの10% */ height: 5vh; /* ビューポート高さの5% */ background-color: lightgoldenrodyellow; }
